## âš™ï¸ Local Installation and Setup

### Prerequisites
* Python 3.x
* Git

### Steps

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Sne44/oceania-gift-shop.git](https://github.com/Sne44/oceania-gift-shop.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd oceania-gift-shop
    ```
3.  **Install dependencies (You would typically run this after creating a virtual environment):**
    *Note: You would use a `requirements.txt` file here, which isn't visible in the files, but this is standard practice.*
    *If you use a virtual environment, activate it here.*
    ```bash
    pip install -r requirements.txt 
    ```
4.  **Apply Migrations:**
    ```bash
    python manage.py migrate
    ```
5.  **Run the development server:**
    ```bash
    python manage.py runserver
    ```
    The application will now be running at `http://127.0.0.1:8000/`.
